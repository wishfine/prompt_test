## 实际共性

1. **送分不等于纯教材原句。** 四选一、常见量估测、属性不变判断、简单电路原理也出现在送分档。共性是不需要重建隐含模型。
2. **基础题允许显性的 2—3 步因果链。** 标准动态电路、杠杆力臂、惯性轨迹和显性报警链并不因“有过程”自动到中等。
3. **中等题有两条路径。** 一类是标准实验、两状态常规比较和公式串联；另一类是无计算但辨析密度高、需反例或充分/必要条件判断的概念题。
4. **拔高的关键是决定性转换，不是特征凑数。** 图线身份反推、切割前后不变量、等效替代、隐含力臂、联立关系和极值是核心卡点。
5. **压轴的关键是整题全链耦合。** 部分真实压轴题在模型输出中只被计为 6—8 步，但其多对象、多状态、多图信息和前问结果复用形成了完整递进链。

## 已修复的机制

- Prompt 主轴改为「直接识别 → 显性应用 → 常规分析 → 决定性转换 → 全链耦合」。
- 删除得分率 `P` 相关表述，因为输入不存在该字段。
- 送分/基础交给 Prompt 语义判断；现有 18 个 features 在这两档重叠过大，不新增 feature-only 降档规则。
- 中等降基础的低结构规则增加「6 个以上高密度概念辨析」出口。
- 新增「综合模型 + 双状态以上 + 图像反推 + 至少两个公式」的决定性转换规则，最多从中等调整到拔高一档。
