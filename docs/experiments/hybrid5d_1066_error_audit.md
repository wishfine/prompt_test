## 3. 抛开 Prompt 后，准确率是否还有提升空间

有，而且主要不是继续堆关键词，而是统一任务计量口径。

对全部 26 道严重偏差和按主要混淆格分层抽取的 72 道相邻误判进行独立复核后，可见：

1. 教师送分/基础边界包含一定主观性。教师“容易”不仅包含纯识记，也包含单一熟悉模板的一步透明应用；Hybrid 对此收得过紧。
2. 基础/中等边界同时存在双向问题。独立选项不能累计为纵向步骤，但完整实验和同一题目特有模型中的多任务负担也不能完全丢弃。
3. 中等/拔高边界是最有把握的提升点。78 道拔高→中等中，大量题确有约 5—6 步综合链、两状态图像、误差方向或完整实验流程，Hybrid 的“必须有决定性转换”过严。
4. 拔高/压轴边界需要放开“模型依赖”，但不能把普通多开关、安全范围自动判压轴。

在不大幅牺牲泛化性的前提下，预计可从以下方向取得净收益：

- 送分/基础：净回收约 15—30 题；
- 基础/中等：净回收约 20—35 题；
- 中等/拔高：净回收约 20—30 题；
- 拔高/压轴：净回收约 5—10 题。

考虑 Lite 温度固定为 1 的随机波动和标签噪声，下一版单次全量合理目标是 64%—67%，而不是承诺直接超过 70%。Prompt 单轮若能达到 64% 以上、严重偏差不超过 20，并保持一档内比例约 97%，即说明结构修复有效。

## 4. 五档边界是否清晰

教师详细分析中的“步数口径”整体有明显层级：在可解析到步数的匹配样本中，容易平均约 1.00 步、较易 1.66、中等 3.51、较难 5.37、困难 6.66；但实际题目与详细分析存在少量错配。因此“文字标准大体清晰，实际执行不完全一致”。

建议统一使用三个内部量：

- 纵向链深 D：最高难单项的前后依赖有效物理决策数；
- 横向任务量 B：整题非重复、非直接检索的有效任务数；
- 高阶结构 H：模型转换、多状态、多对象、独立约束、反推、联立、误差方向、分类、极值、边界验证等。

最终互斥边界：

| 档位 | 必要边界 | 明确排除 |
|---|---|---|
| 送分 | D≤1；单一熟悉模板；无第二次物理决策 | 规律选择、完整模型重构、不同知识目标 |
| 基础 | D=1—2；模型显性；简单应用或多个独立基础任务 | 选项数量和重复判断不能构成中等 |
| 中等 | D=3—4 的连续常规分析；或达到门槛的完整实验/题目特有模型横向任务量 | 仅“知识点多、工作量大、容易错”不够 |
| 拔高 | 决定性转换；或 D≈5—6 且至少两类实质结构共同参与 | 普通多公式、多开关、单一范围词不够 |
| 压轴 | D≥7 且至少三类结构耦合；或 D=5—6、3个核心高阶结构加1个强压轴结构 | 项目、传感器、最大/安全等关键词不够 |

横向任务量最多支持上调一个相邻档，不能单独把基础推到拔高，也不能把中等推到压轴。step_count 始终只记录纵向链深，避免 feature 为了最终等级被抬高。

## 5. 后处理结论

- Hybrid raw：637 道完全一致。
- 原 Hybrid 后处理：656，道净增 19。
- 当前窄规则回放：657，道净增 20，仅比原 Hybrid 多 1 道。
- 扩大“低结构概念题中等→基础”规则的离线试算：约 32 改善、34 恶化，无净收益。

严重偏差不能依靠一条宽泛规则全部消除：生产推理时没有教师标签，只能识别“模型等级与自身 features 明显矛盾”的样本。在此限制下，增加两条仅用于 `hybrid5d_refined` 的窄安全网：

1. `hybrid_medium_to_basic_independent_concept_guard`：原判中等，但为纯文字概念题、0—1 个公式、单状态、无约束、无变量/实验/图表推导，且理由明确承认各选项彼此独立时，降为基础。充分必要条件、反例、边界、规范表述、控制变量等结构会阻止降档；依赖单图的题也不降。
2. `hybrid_medium_to_hard_dense_formula_state_guard`：原判中等，但已同时具有双状态以上、多公式联立和综合物理模型时，升为拔高。

在 Hybrid5d 历史 raw 输出上回放：

| 指标 | 原 Hybrid 最终 | refined 当前规则回放 |
|---|---:|---:|
| 完全一致 | 656 / 61.54% | 663 / 62.20% |
| 相差不超过一档 | 1040 / 97.56% | 1045 / 98.03% |
| MAE | 0.4090 | 0.3977 |
| 严重偏差 | 26 | 21 |

新规则的触发审计：

- 独立概念保护触发 8 道：教师基础 4、送分 3、中等 1；净增 3 道精确命中，并将 3 道送分→中等收缩为一档偏差。
- 多公式双状态保护触发 13 道：教师拔高 7、压轴 2、中等 4；净增 3 道精确命中，并将 2 道压轴→中等收缩为一档偏差。

因此 refined 版本不增加更宽泛的降档或关键词规则，保留已经审计过的相邻一档联合规则：

- 中等→拔高：决定性图像/状态转换；
- 中等→拔高：明确 5 步以上且有综合结构，或多项联合高阶证据；
- 拔高→压轴：多约束和多项联合高阶结构；
- 拔高→压轴递进边界链：保留开关用于消融；
- 低结构概念保护：保留开关，但不再扩大适用范围。

所有后处理最多调整一档，并保留 `difficulty_level_raw` 与 `postprocess_actions`。新安全网可通过 `ENABLE_HYBRID_SEVERE_DEVIATION_GUARDS=0` 关闭，用于新 Prompt 全量重跑时做消融对比。
