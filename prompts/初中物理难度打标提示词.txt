本系统的教师标准对应关系为：容易=送分题，较易=基础题，中等=中等题，较难=拔高题，困难=压轴题。

==================== 一、总原则 ====================

1. 难度来自真实物理推理，不来自题干长度、图片数量、题型名称、题目位置、试卷占比或“最后一题”。
2. 如果输入提供真实得分率 P，按教师区间校准：P≥0.85 为容易，0.70≤P<0.85 为较易，0.50≤P<0.70 为中等，0.25≤P<0.50 为较难，P<0.25 为困难。输入没有真实 P 时，不得估计 P，也不得伪造 P 字段。
3. step_count 只计算真实物理推理、模型转换、公式变形、条件筛选和有效解释；不计算抄写、逐项阅读、画线、接线、重复代入和机械填空。
4. 特征必须如实填写，不能为了迎合最终等级而反向修改特征。

==================== 二、五档标准 ====================

【送分题 / 容易】
- 单个最基础知识点，直接对应教材原文、概念、公式名称、单位或规律；
- 单对象、单状态，无过程拆解；实际有效推理不超过1步；
- 无运算，或只有一步整数算术；不需要公式变形、受力分析、电路分析、实验推理；

- 完整受力图、透镜/平面镜作图、螺线管作图、电路连接和实验装置分析至少为基础题。

【基础题 / 较易】
- 单个知识点简单应用，或同小节内1-2个直接关联知识点；
- 熟悉场景、单对象、单过程，模型完全显性；有效推理约1-2步；
- 可有简单公式代入、简单变形、基础读图、规范作图和基础实验操作。

【中等题】
- 2-3个关联知识点综合；两个连续物理过程，或稍复杂的常规生活/实验情境；
- 需要识别半隐蔽的常规模型；有效推理约3-4步；
- 可有多步代数、简单方程、比例计算或常规图像分析。

【拔高题 / 较难】
- 3个及以上知识点或跨章节综合；多状态、多对象、半陌生情境或简单隐含条件；
- 需要自主建构典型模型和知识迁移；有效推理约5-6步；
- 可涉及2-3个方程联立、比例法、差值法、实验评价、误差方向或几何转化。

【压轴题 / 困难】
- 跨模块深度综合或陌生信息探究；多对象、多过程、多约束；
- 存在临界状态、多解、隐含条件或复杂变量关系；有效推理7步以上；
- 需要等效、分类讨论、极值、不等式、复杂多方程、开放设计、边界覆盖、可行性验证或严密论证。
- 项目、任务、实践、传感器、热敏电阻、压敏电阻、电磁继电器、自动控制等只是情境信号，不能单独决定压轴。

==================== 三、step_count 粗粒度兼容 ====================

保留以下枚举，不得修改：1-2步、3-5步、6-8步、9-12步、12步以上。

- 1-2步：无推理或一步直接识别可判送分；需要1-2步应用判基础。
- 3-5步：实际3-4步通常判中等；接近5步且同时有多状态、跨章节、隐含条件、实验评价、几何转化或方程联立，才考虑拔高。
- 6-8步：通常至少拔高；若实际达到7步以上且至少3个核心高阶信号，并有强压轴信号，才考虑压轴。
- 9-12步、12步以上：只是压轴候选，仍须排除机械步骤堆积、题干很长和重复计算。

==================== 四、多小问、作图和多空 ====================

1. 多问相互独立：以最高难小问为主要依据，但小问数量和知识点数量不能机械累加。
2. 多问层层递进：后问依赖前问结果，并串联读图、计算、解释、评价时，按完整推理链判断整题。
3. 单个知识点的多个同类直接识记空可判送分；两个及以上相互独立知识点，尤其跨不同现象或章节，通常至少基础题。
4. 仅判断重力方向、识别特殊光线名称、补全唯一且完全显性的方向，才可能判送分；完整画出两个及以上力、透镜特殊光线、平面镜成像、螺线管、磁感线或电路连接，至少基础题。
5. 生活应用题不是自动送分：刀刃增压、惯性应用、材料性质、振动频率、热值与扩散等，需要把情境映射到物理规律，通常至少基础题。

==================== 五、项目式和自动控制题 ====================

- 普通工作原理判断：中等或拔高；
- 控制逻辑分析和简单参数筛选：拔高；
- 只有同时存在方案设计、多约束、边界范围、可行性验证、复杂变量关系、分类讨论或严密论证，才考虑压轴；
- “最大、最小、范围、传感器、继电器、项目、装置、安全”等单个词不能触发压轴。
