三、原始正确率标定
================================================================================

目标学生群体是已经完成对应高中物理教学内容、参加普通高考的全体考生，不是竞赛生、重点班学生，也不是尚未学习该内容的学生。

先分别完成两项判断，再选择 accuracy_anchor 并给出 0—100 的连续数值：

1. 局部模型熟悉度：每个局部步骤使用的是教材直接结论、熟悉标准模型、深层模型还是陌生迁移；
2. 整题完成负担：整道题需要完成多少种评分任务、多少次有效物理决策，是否存在多状态传递、多问依赖、模型切换、联合约束，以及关键环节失败是否会造成后续连续失分。

局部方法熟悉不等于整题正确率高。不得仅因所有局部步骤都属于课内标准模型，就把整题原始正确率限制在58以上。最终正确率必须同时反映局部模型熟悉度和整题完成负担，以更能造成真实失分的一侧为主要依据。

accuracy_anchor 只是结构参照，不是隐式难度标签，也不是硬分数桶。不得先选最终难度档，不得从锚点区间抄写72、62、86、89、93等模板分数；必须结合题目实际结构在重叠参考范围内连续估计。

原始正确率应体现每个普通特征自身带来的基础难度，但不要因为“同时出现3个或4个高难特征”再额外施加组合惩罚。多项高难特征共同出现产生的协同放大效应，由程序统一使用 0.85 / 0.70 乘数处理。换言之：
- 单项特征的普通影响：体现在原始 predicted_accuracy 中；
- 多项高难特征的组合乘数效应：不得提前计入原始 predicted_accuracy。

### 1. 高中物理原始正确率结构锚点

这里预测的是普通高考全体考生在已经学完相应内容后的整题正确率，不是“你认为题目有多复杂”的主观分数。不能因为题目属于高中物理，就把教材原型题和一步基础题习惯性压到 70—82。

- accuracy_anchor=教材直接原型，通常 92—96：课本直接结论、直接读图、单一概念判断、关系完全显性的一步代入或标准操作；没有第二次独立物理决策。极少数几乎不存在正常失分路径的题可略高于96。
- accuracy_anchor=低结构基础应用，通常 88—<92：仍是低结构任务，但存在一次简单转换、方向判断或规范操作。多个选项只有在使用同一回答规则时才可归入这一锚点；若分别调用多套知识规则，不得据此锚点直接给出88分以上。
- accuracy_anchor=熟悉标准模型，通常 85—<88：通常只有1—2个有效物理决策，条件显性，无隐藏模型、联合筛选或真实状态依赖。若包含多个异质评分任务，应根据整题负担向下估计，不能只看局部模型熟悉。
- accuracy_anchor=常规综合，典型约65—82：存在真实连续分析、多个关联步骤、普通状态转换或常规综合。若整题负担较轻可以接近85；若包含多个异质任务、6步以上有效决策、递进小问或联合结构，可以低至55左右。
- accuracy_anchor=较长标准综合，典型约45—68：较长的课内标准综合链、多个相互依赖任务或明显整题完成负担。若达到9步以上、包含多个异质评分任务、多阶段连续传递或关键环节会造成后续连续失分，允许低于38；“课内标准模型”不能作为保持58分以上的理由。
- accuracy_anchor=较难综合，典型约25—55：明显模型转换、多状态联合、联合约束、高认知转换或高负担综合。即使方法均来自课内，也可因整题完成负担进入38分以下。
- accuracy_anchor=压轴复杂结构，通常低于35：压轴级强耦合、复杂临界/分类/范围筛选、陌生迁移或长链综合。不得因为题目位置或“压轴”名称直接选择本锚点。

这些范围互相重叠，只提供参考中心，不与最终五档一一对应。相同 accuracy_anchor 可以因整题完成负担不同跨越相邻甚至多个正确率阈值。不得把“常规综合=72”“较长标准综合=62”“较难综合=45”作为固定模板。

特别说明：
- 独立任务不机械增加 step_count，但多个异质评分任务会降低整题完成正确率；
- 普通选择题不能将各选项正确率机械相乘，但多选题若跨多套知识规则，必须考虑完整辨析负担；
- 9步以上、三个及以上异质评分任务、多阶段状态连续传递、多次模型切换、多对象多过程联合，均可打开58分以下甚至38分以下的通道，但任何单项都不是机械定档条件；
- 高难特征少于3个只表示程序不应用组合乘数，不表示原始正确率必须高于58。

### 2. 低于 85 的结构证据

predicted_accuracy < 85 时，boundary_crossing_evidence 必须至少包含一个由 features 和题目实际求解支持的受控编码：

- THREE_PLUS_DEPENDENT_DECISIONS：单个任务本身至少有3个前后依赖的有效物理决策；
- MODEL_NOT_FULLY_EXPLICIT：模型为半隐含、隐含或需要自主建模；
- MULTI_STATE_DEPENDENCY：至少两个状态且前后强依赖或连续变化伴随边界；
- MULTI_RELATION_COMBINATION：多个物理关系需要在同一推理链中联合使用；
- GRAPH_HIDDEN_QUANTITY_INFERENCE：需要从图像反推隐藏量或进行多图联合转换；
- EXPERIMENT_DATA_REASONING：需要标准数据处理、控制变量、故障分析、误差反演或方案设计；
- DEPENDENT_SUBQUESTIONS：后问真实依赖前问结果；
- HIDDEN_CONDITION：实际求解需要识别题干未直接说明的物理条件；
- MULTI_OBJECT_COUPLING：多个对象需要分别建模并双向耦合或共同受约束；
- JOINT_CONSTRAINTS：多个约束共同筛选同一解或范围；
- CONCEPTUAL_MODEL_CONFLICT：概念题需要围绕同一物理模型联合辨析多个相互关联、反直觉的关系，例如场强、电势、电势能和电荷正负的关系。必须在 reason 中写明具体冲突，不能只写“容易混淆”。

“选项多”“知识点多”“逐项判断”“整体容易错”“符号可能错”“方向容易错”“一次单位换算”“题目带图”不是合法的跨越85分证据。

### 3. error_risk 的使用边界

error_risk 不负责决定正确率所属的大区间，只负责在已经确定的结构锚点区间内微调。

一题若为“模型完全显性、1—2步、直接套用或简单因果”，即使存在方向、符号、单位、概念名称等普通易错点，也应优先留在其 85 分以上的结构区间，不能仅凭“容易混淆”降到72或78。

只有易错点背后真实对应新的物理决策、隐藏条件、多概念联合辨析、模型转换或约束筛选时，才能选择更低的结构锚点。error_risk 不能与 reasoning_chain、hidden_conditions、classification_discussion 等字段对同一个难点重复惩罚。

### 4. 输出前正确率自检

- 若 predicted_accuracy < 88，below_88_justified 必须反映是否存在不能归入“低结构基础应用及以上”的真实依据。
- 若 predicted_accuracy < 85，below_85_justified 必须为 true，且 boundary_crossing_evidence 必须包含至少一个真实成立的受控编码。
- 普通选择题不得将选项当作独立任务相乘，options_treated_as_independent_tasks 必须为 false。
- error_risk 只能区间内微调，error_risk_only_used_for_local_adjustment 应为 true。
- “没有程序定义的高难特征”只表示不触发组合乘数，不代表题目一定简单；多步骤标准综合仍可能落在较低原始正确率区间。
- 题目长、图多、知识点名称陌生、出现连接体或守恒定律，均不能单独降低正确率。
- 若选择“教材直接原型”“低结构基础应用”或“熟悉标准模型”，必须检查是否遗漏了多个异质知识规则、实验评分点、图像转换或分别要求作答的输出。
- 若题目达到9步以上，或同时存在三个及以上过程、三个及以上状态和多问依赖，却仍给出58分以上，必须在 reason 中说明为什么整题完成负担仍不足以进入更低区间。
- 若题目使用课内标准模型但存在长链、多状态、多问依赖或联合约束，必须依据整题完成负担估分，不能使用“方法常规”作为高分理由。

程序稍后会用乘数后的 predicted_accuracy 按以下连续区间映射档位：
- predicted_accuracy >= 88：难度1档
- 85 <= predicted_accuracy < 88：难度2档
- 58 <= predicted_accuracy < 85：难度3档
- 38 <= predicted_accuracy < 58：难度4档
- predicted_accuracy < 38：难度5档

你当前输出的 predicted_accuracy 仍是乘数前的原始值。高难结构的额外乘数由程序统一执行。
